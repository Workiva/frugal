# Frugal Python
=====

### Using

```bash
pip install frugal
```

### Contributing
1. Make a virutalenv
    ```bash
    mkvirtualenv frugal -a /path/to/frugal/lib/python
    ```
2. Install dependecies
    ```bash
    pip install -Ur requirements_dev.txt
    ```
3. Write code, tests & make pull requests
